## Overview

This project is a simple e-commerce application for selling perfumes
online. It allows users to browse products, view details, add items to
cart, and complete purchases using PayPal integration.

The goal of this project was to build a clean and functional online
store while integrating a real payment gateway.

------------------------------------------------------------------------

## Features

-   Product listing and detail pages
-   Add to cart functionality
-   Order summary and checkout flow
-   PayPal payment integration
-   Order confirmation after successful payment
-   Clean and structured project architecture

------------------------------------------------------------------------

## Payment Integration

-   Secure PayPal checkout
-   Payment verification after transaction
-   Sandbox testing environment for development

------------------------------------------------------------------------

## Tech Stack

-   Backend: Django / DRF / FastAPI  
-   Frontend: React  
-   Payment Gateway: PayPal API
-   Database: PostgreSQL 

------------------------------------------------------------------------

## Purpose

This project was built to: - Understand e-commerce workflow
implementation - Integrate third-party payment gateways (PayPal) -
Practice order management and transaction handling - Build a structured
full-stack application
